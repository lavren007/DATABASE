## Описание баз данных

### 1. Транспортные средства (`transport_db`)
Моделирует учёт транспортных средств трёх типов: автомобили, мотоциклы и велосипеды.  
**Задачи:**
- Найти мотоциклы с мощностью >150 л.с., ценой <20000 $, типа Sport.
- Объединённая выборка по автомобилям, мотоциклам и велосипедам с заданными фильтрами.

[Подробнее → transport_db/README.md](transport_db/README.md)

### 2. Автомобильные гонки (`racing-db`)
Содержит информацию о классах автомобилей, самих автомобилях, гонках и результатах.  
**Задачи (5 шт.):**
- Для каждого класса определить автомобили с наименьшей средней позицией.
- Найти автомобиль с глобально лучшей средней позицией.
- Определить классы с наименьшей средней позицией.
- Выявить автомобили, результаты которых лучше среднего по классу.
- Найти классы с наибольшим количеством автомобилей, имеющих среднюю позицию >3.0.

[Подробнее → racing-db/README.md](racing-db/README.md)

### 3. Бронирование отелей (`hotel-booking-db`)
Управляет бронированиями в отелях.  
**Задачи (3 шт.):**
- Клиенты с более чем двумя бронированиями в разных отелях.
- Клиенты с >2 бронированиями и тратами >500 $.
- Классификация предпочтений клиентов по типу отелей.

[Подробнее → hotel-booking-db/README.md](hotel-booking-db/README.md)

### 4. Структура организации (`organization-db`)
Отражает иерархию сотрудников, отделов, проектов и задач.  
**Задачи (3 шт.):**
- Рекурсивный вывод всех подчинённых Ивана Иванова.
- То же с добавлением количества задач и прямых подчинённых.
- Список менеджеров, имеющих подчинённых, с общим количеством подчинённых по иерархии.

[Подробнее → organization-db/README.md](organization-db/README.md)

## Общие требования

- Все SQL-скрипты написаны для **PostgreSQL** (версии 12 и выше) и совместимы с `SERIAL` для автоинкремента.
- Каждая база данных содержит:
  - скрипт создания таблиц (`schema.sql`);
  - скрипт заполнения тестовыми данными (`data.sql`);
  - отдельные файлы для каждого задания;
  - документацию `README.md` с подробным описанием структуры и инструкцией по запуску.

## Инструкция по развёртыванию (общая)

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd DATABASES
   ```

2. Для каждой базы данных выполните последовательно:
    - создайте отдельную базу данных (например, `transport_db`, `racing_db` и т.д.);
    - выполните скрипт создания таблиц;
    - выполните скрипт наполнения данными;
    - выполните файлы задач для получения результатов.

   Более детальные шаги описаны в README каждой подпапки.
